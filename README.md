# Sentiment & Opinion Mining Natural Language API Hackathon

## Introduction - Employee Monitor

Employee Monitor allows, thanks to Expert.ai, to analyze the comments of the employees in a company, and obtain the sentiment and keywords (topics) so that human resources can take faster actions to maintain the satisfaction of all its collaborators.

## Technologies

- Visual Studio 2017 or superior.
- C#
- ASP.NET MVC 5 or superior
- SQL Server 

## Configuration

In Web.config, you must adjust the connection strings (add the hostname, user, password and name of the database).
Also, in Web.config you must change the keys with your Expert.AI username and password for executing.
